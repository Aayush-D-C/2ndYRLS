CSS Table Properties

1. border-collapse

Collapses or separates table borders. Below is a table with border-collapse: collapse.

2. border-spacing

Specifies the spacing between the borders of cells. Below is a table with border-spacing: 10px 5px.

3. caption-side

Positions the table caption. Below is a table with caption-side: bottom.

Controls whether empty cells should have borders/backgrounds. Below is a table with empty-cells: hide.
